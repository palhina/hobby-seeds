## データ設計例

### ゆる趣味（YuruHobby）

```json
{
  "id": 1,
  "name": "窓の外を眺める",
  "emoji": "🪟",
  "category": "眺める",
  "tags": ["リラックス", "静か", "一人"],
  "time": 5,
  "cost": 0,
  "location": "家",
  "energy": "low",
  "indoor": true,
  "tryStep": "窓際に座って、5分だけ外の景色を眺めてみる"
}
```

### ステップアップ趣味（StepUpHobby）

```json
{
  "id": 101,
  "name": "写真撮影",
  "emoji": "📷",
  "tags": ["クリエイティブ", "記録"],
  "matchTags": ["眺める", "散歩", "自然"],
  "description": "スマホでも始められる手軽な趣味",
  "startCost": "0円（スマホ）〜",
  "startGuide": "散歩中に気になったものを撮ってみる",
  "timeCommit": "週に1-2時間",
  "nextSteps": [
    "SNSに投稿してみる",
    "編集アプリを試す",
    "カメラ入門書を読む"
  ]
}
```

## カテゴリ別アイデア

### 眺める 👀
- 窓の外を眺める
- 雲を観察する
- 水族館の動画を見る
- 炎を見る（キャンドル）
- 星空アプリを使う

### 作る 🎨
- 落書きをする
- 折り紙を1つ折る
- 写真を1枚加工する
- メモ帳に何か書く
- お菓子のパッケージで何か作る

### 動く 🏃
- ストレッチを1種類
- 家の中を歩く
- ベランダに出る
- 近所を5分散歩
- ラジオ体操1セット

### 聴く 🎧
- 好きな曲を1曲聴く
- 自然音を流す
- ポッドキャストを5分
- ASMRを聴く
- 環境音楽をBGMに

### 学ぶ 📚
- Wikipediaをランダムに
- TED Talkを1本
- 雑学クイズアプリ
- 今日の天気を調べる
- 昔の写真を見返す

### 整える 🧹
- 机の上を片付ける
- 引き出し1つを整理
- スマホの写真を整理
- 財布の中を整理
- メールを5件整理

### 遊ぶ 🎮
- パズルゲーム1問
- トランプで一人遊び
- 紙飛行機を飛ばす
- シャボン玉を作る
- ボールを投げる
